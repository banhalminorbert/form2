2. Verarbeitete Daten

Diese Website setzt keine Cookies, verwendet keine Analyse, lädt keine externen Schriftarten, bindet keine Karten oder Chat-Widgets ein und enthält keine Marketing- oder Tracking-Pixel Dritter.
